# Cytech EC サイト

## 概要
Spring Bootで開発したECサイトのポートフォリオです。

## 使用技術
- Java 17
- Spring Boot 4.0
- Spring Security
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven
- CSS

## 機能一覧
- ユーザー登録・ログイン・ログアウト（Spring Security）
- 商品一覧・商品詳細表示
- カート機能（追加・削除・一覧）
- 購入機能（カートから購入・単品購入）
- 住所・支払い方法の入力
- 口コミ投稿・表示（画像アップロード対応）
- マイページ（ユーザー情報・注文履歴）
- 商品検索（キーワード・カテゴリ）
- バリデーション
- TOP画面セール品表示
- レスポンシブデザイン
